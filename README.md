# mongo-db-docker
Mongo DB with docker

## Params
To test, change this params in run.sh file.

| Params  | Descriptions |
| ------------- | ------------- |
| LOCAL_PORT | Local port connect to DB (Through docker)  |
| LOCAL_DIR_DB_DATA  | To save information  |
## Installation
```bash
sudo chmod +x run.sh
./run.sh
```
